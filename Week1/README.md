## 📄 Report Contents

The report is structured across **7 sections**:

1. **Introduction to Cybersecurity** — Fundamentals, importance, and current threat relevance
2. **5 Major Modern Cyber Threats** — In-depth analysis of each threat
3. **Impact Analysis** — Individual and organizational impact matrix
4. **Real-World Case Studies** — One incident per threat
5. **Preventive Measures** — Technical and procedural defenses
6. **Conclusion & Future Scope** — Emerging threats and continuous learning
7. **References** — Cited sources from CISA, IBM, Verizon DBIR, ENISA, and more
